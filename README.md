# 🇮🇳 Indian State Guessing Game

An interactive Python game where users guess the names of Indian states and see them displayed on a map.

## 🚀 Features
- Real-time state guessing
- Dynamic mapping using Turtle
- Data handling using CSV
- Interactive learning experience

## 🛠️ Tech Stack
- Python
- Turtle
- CSV

## ▶️ How to Run
1. Install Python
2. Run:
   python main.py
